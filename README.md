### Run in Unity
1. Clone the repo.  
2. Open in Unity.  
3. Open `Assets/Scenes/XXXX.unity`.  
4. Run the scene.  

### GameBuild
1. Clone the repo.  
2. Open in Unity.  
3. Go to **File > Build Settings**.  
4. Make sure all scenes are included.
5. Verify build settings [here]).
6. Click **Build** and choose save location.
